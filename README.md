## Prerequisites

- npm >=5.5.0
- node >=9.3.0
- npm install @top-gg/sdk
- npm install dbl.js
- npm install dblapi.js
- npm install discord.js
- npm install fs
- npm install random-puppy
    

## Usage

- node index.js
